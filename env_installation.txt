Installationsanleitung für eine env, womit das Projekt unter Windows ausgeführt werden kann
1)
    vsBuildTools runterladen und C++ Desktop Entwcklungsumgebung und .Net BuildTools installieren
    -> https://visualstudio.microsoft.com/downloads/?q=build+tools
    -> https://www.youtube.com/watch?v=zRaDe08cUIk @10:40
2)
    espeakng runterladen/installieren
    -> https://github.com/espeak-ng/espeak-ng/blob/master/docs/guide.md#windows
3)
    cuda toolkit 10.1 runterladen/installieren
    -> https://developer.nvidia.com/cuda-10.1-download-archive-base
    danach Grafikkartentreiber aktualisieren
    evtl. muss man sich als developer registrieren bei nVidia (dauert 10 Minuten)
4)
    conda env mit der environment.yaml aus diesem repo erstellen
    davor in der letzten Zeile den Pfad, in dem das env installiert werden soll ändern bzw. auf seine location anpassen,
    wo die eigenen env's liegen
    conda env create -f environment.yaml
    * bevor Schritt 5 abgeschlossen werden kann und das env steht müssen die Schritte 1 bis 4 komplett abgeschlossen
    werden, sonst failed entweder die installation von tts oder es gibt später beckenderrors im Code o.ä.

    * Referenz -> https://stackoverflow.com/questions/66726331/how-can-i-run-mozilla-tts-coqui-tts-training-with-cuda-on-a-windows-system

    * Vielleicht könnte man noch ein installation script erstellen
